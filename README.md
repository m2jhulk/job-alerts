Project Overview:
Sarkari Result Email Alert System is a completely serverless automation web application. It features a custom, responsive frontend subscription form that securely connects to a free backend database using Google Apps Script and Google Sheets API.

Key Features:

Designed to help aspirants automatically track and receive real-time email notifications for government job updates, admit cards, and major exam results (such as UGC-NET, DDA, and Bank Officer roles).

Built without any paid backend server, ensuring 100% free hosting and database management.

Implements duplicate email prevention and basic spam protection at the API level.

Tech Stack Used: HTML5, CSS3, JavaScript (Fetch API), Google Apps Script, Google Sheets.
